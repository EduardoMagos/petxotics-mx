¿Qué es SiCenapyme?

SiCenapyme es un sistema web montado en el servidor de la Facultad de Contaduría y Administración, fue elaborado con el proposito de ayudar con la gestión de cursos, talleres y diplomados que ofrece Cenapyme, además de administrar y validar las inscripciones de los eventos antes mencionados.


Este sistema fue desarrollado con la metodología de desarrollo ágil llamada SCRUM, se programo bajo el paradigma de la programación orientada a objetos y por último se utilzó el Modelo - Vista - Controlador (MVC) como patrón de arquitectura de software.



¿Qué hice yo en este proyecto?

A grandes rasgos yo participe durante todo el desarrollo de este proyecto, desde el diseño de la base de datos hasta la implementación en el servidor. Pero siendo concretos los puntos en los que participé serían los siguientes:


   Obtención de requerimientos

Entreviste al cliente para conocer sus necesidades o problematicas y así establecer una solución sastisfactoria que estuviera en los alcances del proyecto.


   Diseño e implementación de la base de datos

De acuerdo con las reglas del negocio, requerimientos y con arquitectura - estándares del servidor diseñamos y desarrollamos la base de datos, durante el proceso del diseño nos auxiliamos de la herramienta PowerDesigner, por último, como manejador de bases de datos utilizamos PostgreSQL.


   Diseño web y prototipado

Aquí desarrollamos dos tipos de prototipos, el interactivo desarrollado con Justinmind y el estático desarrollado con Adobe XD, la diferencia es que el primero era para ver el flujo del sistema y el segundo era para abarcar la parte del diseño y UX.


   Programación

En este proyecto utlizamos principalmente el lenguaje de programación PHP para la cuestión de definición de clases y controladores, para el manejo de la información entre las vistas y los controladores utilizamos JavaScript anexando la implementación de Ajax. Para las vistas utilizamos HTML y CSS con ayuda del framework llamado Bootstrap, por último, para la validación de los formularios utilizamos igualmente JavaScript.
Un requerimiento interesante de este proyecto era generar las listas y gafetes de los alumnos correctamente inscritos a los eventos de Cenapyme, dicho requerimiento lo solucionamos con la implementación de la biblioteca fpdf.
